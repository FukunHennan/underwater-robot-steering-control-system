# 文档目录

此文件夹用于存放项目相关的 Markdown 文档文件。

## 目录结构

| 文件名 | 描述 |
|--------|------|
| MODBUS_DOC.md | Modbus 通信协议文档 |
| 硬件系统流程图.md | 硬件系统架构和流程图文档 |

## 文档规范

1. **文件名**：使用描述性名称，英文文档使用小写字母加下划线，中文文档使用中文名称
2. **格式**：使用 Markdown 格式，遵循标准 Markdown 语法
3. **内容**：
   - 包含文档标题和版本信息
   - 提供清晰的章节结构
   - 使用表格和代码块提高可读性
   - 包含必要的图表和示意图

## 如何添加新文档

1. 将新的 Markdown 文件放入此文件夹
2. 在本 README.md 中更新文件列表
3. 确保文档内容完整且格式规范

## 版本控制

所有文档文件都应纳入版本控制，确保文档与代码同步更新。
