# 水下智能转向系统 — 设备功能清单

> 毕业论文参考文档 · 作者: chenfukun · 2026

---

## 一、系统概述

本系统是一套面向水下机器人的智能转向控制平台，以 **STM32F407VET6** 为主控核心，集成姿态感知、多路执行器驱动、环境传感器采集和工业标准 Modbus RTU 通信，并配套基于 Web 技术的上位机监控仪表盘，实现对水下机器人的远程监测与控制。

| 项目 | 参数 |
|------|------|
| 主控芯片 | STM32F407VET6, ARM Cortex-M4, 168 MHz, 512 KB Flash, 192 KB SRAM |
| 固件版本 | V3.0 (设备ID: 0x0407) |
| 运行模式 | 待机 / 手动 / 自动 |
| 通信协议 | Modbus RTU (USART2, 9600 8N1, Slave Addr 0x01) |
| 上位机 | React + Vite + Tailwind CSS, Web Serial API |

## 二、硬件功能模块

### 2.1 电源系统

| 模块 | 芯片 | 功能 |
|------|------|------|
| 降压电源 | LM2596S-ADJ | 为舵机/电调提供可调电压 |
| 稳压电源 | AMS1117-3.3 | 为主控/传感器提供 3.3V |
| 电压检测 | LM321 运放 + 分压电阻 | 检测电源电压，分压比 3:1，ADC 通道 PA1 |

### 2.2 姿态传感器模块 (ATK-MS901M)

| 项目 | 说明 |
|------|------|
| 型号 | 正点原子 ATK-MS901M 九轴姿态传感器 |
| 接口 | USART3, 115200 baud |
| 输出数据 | 姿态角 (Roll/Pitch/Yaw)、角速度 (Gyro X/Y/Z)、加速度、磁力计、气压计 |
| 融合算法 | 模块内置卡尔曼滤波姿态解算 |
| 数据格式 | IEEE 754 float (Roll/Pitch/Yaw: 度; Gyro: °/s) |
| 采样周期 | 5ms (主循环非阻塞读取) |
| 寄存器存储 | 每个浮点值占 2 个 Modbus 寄存器 (Big-Endian 字序) |

**支持的数据类型:**
- 姿态角 (Roll, Pitch, Yaw)
- 陀螺仪三轴角速度 (Gyro X/Y/Z)
- 加速度计三轴 (Accel X/Y/Z)
- 四元数 (Q0/Q1/Q2/Q3)
- 磁力计三轴
- 气压计 (气压 Pa、海拔 cm、温度 °C)

### 2.3 PWM 执行器控制 (10 通道)

**舵机/电调通道 (8 路, 50 Hz):**

| 通道 | 引脚 | 定时器 | 脉宽范围 | 默认值 |
|------|------|--------|----------|--------|
| Servo 1 | PD13 | TIM4_CH2 | 500–2500 μs | 1500 μs |
| Servo 2 | PD15 | TIM4_CH4 | 500–2500 μs | 1500 μs |
| Servo 3 | PD12 | TIM4_CH1 | 500–2500 μs | 1500 μs |
| Servo 4 | PD14 | TIM4_CH3 | 500–2500 μs | 1500 μs |
| Servo 5 | PC6 | TIM8_CH1 | 500–2500 μs | 1500 μs |
| Servo 6 | PC7 | TIM8_CH2 | 500–2500 μs | 1500 μs |
| Servo 7 | PB0 | TIM3_CH3 | 500–2500 μs | 1500 μs |
| Servo 8 | PB1 | TIM3_CH4 | 500–2500 μs | 1500 μs |

**LED 灯板通道 (2 路):**

| 通道 | 引脚 | 定时器 | 占空比范围 |
|------|------|--------|------------|
| LED 1 | PE9 | TIM1_CH1 | 0–1000 |
| LED 2 | PE11 | TIM1_CH2 | 0–1000 |

- PWM 频率: 50 Hz (周期 20 ms, ARR=19999, PSC=83, 时钟 84 MHz)
- Modbus 写入寄存器后立即通过 `apply_register()` 生效

### 2.4 ADC 数据采集 (5 通道)

| 通道 | 引脚 | ADC通道 | 用途 | 说明 |
|------|------|---------|------|------|
| CH0 (ANALOG1) | PC3 | ADC1_IN13 | 模拟信号采集 | 通用模拟输入 |
| CH1 (ANALOG2) | PC1 | ADC1_IN11 | 模拟信号采集 | 通用模拟输入 |
| CH2 (ANALOG3) | PC2 | ADC1_IN12 | 模拟信号采集 | 通用模拟输入 |
| CH3 (ANALOG4) | PC0 | ADC1_IN10 | 模拟信号采集 | 通用模拟输入 |
| CH4 (VOLTAGE) | PA1 | ADC1_IN1 | 电源电压检测 | 分压比 3:1 |

**ADC 参数:**
- 分辨率: 12 位 (0–4095)
- 参考电压: 3.3V
- 采样时间: 480 周期
- 多次采样取平均 (10 次)

**电压检测通道 (CH4):**
- `V_power = V_adc × 3.0 (分压比)`
- 寄存器存储: 电压 × 100 (uint16)

**模拟输入通道 (CH0–CH3):**
- 采集外部模拟信号, 12 位原始值 (0–4095)
- 寄存器同时存储转换数据和 ADC 原始值

### 2.5 GPIO 扩展接口 (4 路)

| 编号 | 引脚 | 模式 |
|------|------|------|
| GPIO0 | PB12 | 输入/输出 |
| GPIO2 | PE6 | 输入/输出 |
| GPIO3 | PE5 | 输入/输出 |
| GPIO4 | PC4 | 输入/输出 |

- 支持输入读取和输出控制
- 可用于红外对管障碍物检测等外设扩展

## 三、通信协议功能

### 3.1 Modbus RTU 从站

| 项目 | 参数 |
|------|------|
| 协议 | Modbus RTU |
| 物理接口 | USART2 (RS-485 / TTL) |
| 波特率 | 9600 |
| 数据格式 | 8 数据位, 无校验, 1 停止位 (8N1) |
| 从站地址 | 0x01 |
| 协议栈 | FreeModbus |

**支持的功能码:**

| 功能码 | 名称 | 用途 |
|--------|------|------|
| 0x03 | 读保持寄存器 | 读取所有传感器数据和系统状态 |
| 0x06 | 写单个寄存器 | 控制单个舵机/LED/运行模式 |
| 0x10 | 写多个寄存器 | 批量控制多个舵机 |

### 3.2 寄存器地址映射 (共 58 个)

**系统寄存器 (0x0000–0x0005, 6 个):**

| 地址 | 名称 | 类型 | 权限 | 说明 |
|------|------|------|------|------|
| 0x0000 | DEVICE_ID | uint16 | R | 设备标识, 固定 0x0407 |
| 0x0001 | FW_VERSION | uint16 | R | 固件版本, 0x0300 = V3.0 |
| 0x0002 | RUN_MODE | uint16 | R/W | 运行模式: 0=待机, 1=手动, 2=自动 |
| 0x0003 | FAULT_CODE | uint16 | R | 故障码位图 |
| 0x0004 | SYS_TICK_L | uint16 | R | 系统运行时间低 16 位 (ms) |
| 0x0005 | SYS_TICK_H | uint16 | R | 系统运行时间高 16 位 (ms) |

**姿态寄存器 (0x0010–0x001B, 12 个, IEEE 754 float):**

| 地址 | 名称 | 类型 | 权限 | 单位 |
|------|------|------|------|------|
| 0x0010–0x0011 | ROLL | float (2 regs) | R | 度 (°) |
| 0x0012–0x0013 | PITCH | float (2 regs) | R | 度 (°) |
| 0x0014–0x0015 | YAW | float (2 regs) | R | 度 (°) |
| 0x0016–0x0017 | GYRO_X | float (2 regs) | R | °/s |
| 0x0018–0x0019 | GYRO_Y | float (2 regs) | R | °/s |
| 0x001A–0x001B | GYRO_Z | float (2 regs) | R | °/s |

> 浮点存储格式: Big-Endian 字序, 高字在低地址 `[H]<<16 | [L]`

**PWM 控制寄存器 (0x0020–0x0029, 10 个):**

| 地址 | 名称 | 类型 | 权限 | 范围 |
|------|------|------|------|------|
| 0x0020–0x0027 | SERVO1–8 | uint16 | R/W | 500–2500 μs |
| 0x0028–0x0029 | LED1–2 | uint16 | R/W | 0–1000 |

**ADC 采集寄存器 (0x0030–0x0039, 10 个):**

| 地址 | 名称 | 类型 | 权限 | 说明 |
|------|------|------|------|------|
| 0x0030–0x0033 | ANALOG1–4 | int16 | R | 模拟量数据 |
| 0x0034 | VOLTAGE | uint16 | R | 电压 × 100 (V) |
| 0x0035–0x0039 | ADC_RAW0–4 | uint16 | R | 12 位原始值 (0–4095) |

### 3.3 写保护机制

- 只有 `RUN_MODE` 和 `SERVO1–8` / `LED1–2` 寄存器允许写入
- 写入只读寄存器返回 Modbus 异常码
- PWM 寄存器写入后立即执行 `apply_register()` 更新硬件输出

## 四、软件架构功能

### 4.1 固件主循环

```
初始化 → ADC → PWM → MS901M → Modbus → 主循环
                                           ├─ 读取 MS901M 姿态 (5ms 超时)
                                           ├─ 写入姿态浮点寄存器
                                           ├─ modbus_process()
                                           │   ├─ modbus_update_sensors() → 更新模拟量/电压/系统时钟寄存器
                                           │   └─ eMBPoll() → 处理 Modbus 请求
                                           └─ delay 5ms
```

### 4.2 UART 端口分配

| UART | 波特率 | 功能 |
|------|--------|------|
| USART1 | 115200 | 调试串口 (printf) |
| USART2 | 9600 | Modbus RTU 从站 |
| USART3 | 115200 | ATK-MS901M 姿态传感器 |

### 4.3 定时器资源分配

| 定时器 | 用途 | 通道数 |
|--------|------|--------|
| TIM1 | LED PWM | 2 (PE9, PE11) |
| TIM3 | 舵机 7–8 PWM | 2 (PB0, PB1) |
| TIM4 | 舵机 1–4 PWM | 4 (PD12–PD15) |
| TIM8 | 舵机 5–6 PWM | 2 (PC6, PC7) |

## 五、上位机监控系统功能

### 5.1 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 18.x | UI 框架 |
| Vite | 6.x | 构建工具 |
| TypeScript | 5.x | 类型安全 |
| Tailwind CSS | 3.x | 样式框架 |
| Web Serial API | — | 浏览器直连串口 |
| Lucide React | — | 图标库 |

### 5.2 功能模块

| 模块 | 功能描述 |
|------|----------|
| **串口连接** | 通过 Web Serial API 选择 COM 口, 9600 8N1 连接 |
| **自动轮询** | 支持 200ms / 500ms / 1s / 2s 间隔, 顺序读取全部寄存器组 |
| **系统状态面板** | 显示设备ID、固件版本、运行模式、故障码、运行时间 |
| **姿态数据面板** | 显示 Roll/Pitch/Yaw (°) 和 Gyro X/Y/Z (°/s) |
| **3D 姿态可视化** | CSS 3D 变换模型, 实时响应三轴旋转, 标注 TOP/BTM/FWD/AFT/L/R |
| **PWM 舵机控制** | 8 路滑条 (500–2500 μs), 拖动显示即时值, 释放时写入寄存器 |
| **LED 控制** | 2 路亮度滑条 (0–1000), 释放时写入 |
| **ADC 原始数据** | 显示 4 通道 12 位原始值 + 进度条, 电压通道显示转换值 |
| **通信日志** | 底部可展开面板, 显示 TX/RX 帧十六进制数据 + 时间戳 |
| **寄存器速查表** | 显示关键寄存器地址、类型、权限和当前值 |

### 5.3 Modbus 客户端关键设计

| 设计 | 说明 |
|------|------|
| **事务队列** | 异步 FIFO 队列, 保证同一时刻只有一个 Modbus 帧在总线上 |
| **帧间延时** | 每次事务后等待 15ms (> T3.5 @ 9600 baud ≈ 4ms) |
| **CRC16 校验** | 查表法, 与 FreeModbus 一致 |
| **IEEE 754 转换** | `DataView` 实现双寄存器浮点数解析 (Big-Endian 字序) |
| **滑条防抖** | 拖动仅更新本地 UI, 松开鼠标才发送写入, 避免总线拥塞 |
