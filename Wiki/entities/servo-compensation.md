# 舵机姿态补偿

## 功能说明

每路舵机可根据当前姿态角自动调整输出角度，实现水下姿态自稳。

---

## 补偿公式

```
目标角度 = BASE_ANGLE + K_ROLL × Roll + K_PITCH × Pitch + K_YAW × Yaw
PWM脉宽  = 1500 + 目标角度 × 10  (μs)
```

| 系数 | 说明 |
|------|------|
| **BASE_ANGLE** | 基础角度偏移 (deg)，舵机零点校正 |
| **K_ROLL** | Roll 补偿系数，控制横滚方向响应强度 |
| **K_PITCH** | Pitch 补偿系数，控制俯仰方向响应强度 |
| **K_YAW** | Yaw 补偿系数，控制偏航方向响应强度 |

---

## 寄存器地址

每路舵机占用 8 个寄存器（4个float），基址按舵机编号递增 8：

| 舵机 | 基址 | BASE | K_ROLL | K_PITCH | K_YAW |
|------|------|------|--------|---------|-------|
| 舵机1 | 0x00A0 | 0x00A0 | 0x00A2 | 0x00A4 | 0x00A6 |
| 舵机2 | 0x00A8 | 0x00A8 | 0x00AA | 0x00AC | 0x00AE |
| 舵机3 | 0x00B0 | ... | | | |
| 舵机8 | 0x00D8 | 0x00D8 | 0x00DA | 0x00DC | 0x00DE |

启用标志: `SERVO_COMP_ENABLE_n` (0x00E0–0x00E7)，写 1 启用，写 0 禁用。

保存到 Flash: 写 `SERVO_SAVE_CMD = 0x5A5A` (0x009F)。

---

## 使用方法

1. 在上位机**舵机页面**打开对应舵机的补偿开关
2. 调整 BASE_ANGLE 校正舵机零点
3. 设置 K_ROLL/K_PITCH/K_YAW 系数（正值表示正向补偿，负值反向）
4. 观察姿态变化时舵机是否正确跟随
5. 点击**保存到 Flash** 持久化参数

---

## 注意事项

- 补偿输出叠加在上位机发送的 PWM 指令之上，最终脉宽仍限制在 500–2500μs
- 系数过大可能导致舵机振荡，建议从小值（0.1）开始调节
- 启用补偿后，上位机的 PWM 手动控制作为基准值继续有效
