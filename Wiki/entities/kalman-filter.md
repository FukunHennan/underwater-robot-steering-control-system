# 卡尔曼滤波算法

基于 STM32F407 的六通道卡尔曼滤波实现，用于 MS901M 九轴姿态传感器数据平滑处理。

---

## 概述

卡尔曼滤波是一种最优线性估计算法，通过预测-更新两步循环，融合系统模型和传感器测量值，有效降低噪声干扰。

**滤波通道**：Roll, Pitch, Yaw, GyroX, GyroY, GyroZ

---

## 算法原理

### 状态方程

```
预测：
x̂(k|k-1) = A · x̂(k-1|k-1) + B · u(k)
P(k|k-1) = A · P(k-1|k-1) · Aᵀ + Q

更新：
K(k) = P(k|k-1) · Hᵀ / (H · P(k|k-1) · Hᵀ + R)
x̂(k|k) = x̂(k|k-1) + K(k) · (z(k) - H · x̂(k|k-1))
P(k|k) = (1 - K(k) · H) · P(k|k-1)
```

### 本系统简化模型

由于姿态传感器已内置卡尔曼滤波输出欧拉角，本系统的滤波主要用于进一步平滑：

| 参数 | 值 | 说明 |
|------|-----|------|
| 状态矩阵 A | 1 | 一阶恒速模型 |
| 观测矩阵 H | 1 | 直接观测 |
| 过程噪声 Q | 0.001 | 系统模型不确定性 |
| 测量噪声 R | 0.1 | 传感器噪声方差 |

## 数据结构

```c
typedef struct {
    float x;          // 状态估计值
    float p;          // 估计误差协方差
    float q;          // 过程噪声协方差
    float r;          // 测量噪声协方差
    float k;          // 卡尔曼增益
} kalman_t;
```

---

## 核心代码

### 初始化

```c
void kalman_init(kalman_t *k, float q, float r, float p_init)
{
    k->x = 0.0f;
    k->p = p_init;
    k->q = q;
    k->r = r;
    k->k = 0.0f;
}
```

### 滤波更新

```c
float kalman_update(kalman_t *k, float measurement)
{
    // 预测
    k->p = k->p + k->q;
    
    // 更新
    k->k = k->p / (k->p + k->r);
    k->x = k->x + k->k * (measurement - k->x);
    k->p = (1.0f - k->k) * k->p;
    
    return k->x;
}
```

### 六通道实例

```c
static kalman_t kalman_roll, kalman_pitch, kalman_yaw;
static kalman_t kalman_gyro_x, kalman_gyro_y, kalman_gyro_z;

void kalman_init_all(void)
{
    kalman_init(&kalman_roll, 0.001f, 0.1f, 1.0f);
    kalman_init(&kalman_pitch, 0.001f, 0.1f, 1.0f);
    kalman_init(&kalman_yaw, 0.001f, 0.1f, 1.0f);
    kalman_init(&kalman_gyro_x, 0.001f, 0.1f, 1.0f);
    kalman_init(&kalman_gyro_y, 0.001f, 0.1f, 1.0f);
    kalman_init(&kalman_gyro_z, 0.001f, 0.1f, 1.0f);
}
```

## 参数调优

### Q 值 (过程噪声) 影响

| Q 值 | 响应速度 | 平滑度 | 适用场景 |
|------|---------|--------|---------|
| 0.0001 | 慢 | 极高 | 静态测量 |
| 0.001 | 中 | 高 | 水下转向 (推荐) |
| 0.01 | 快 | 中 | 动态跟踪 |
| 0.1 | 极快 | 低 | 快速响应 |

### R 值 (测量噪声) 影响

| R 值 | 信任传感器 | 平滑度 | 适用场景 |
|------|-----------|--------|---------|
| 0.01 | 高 | 低 | 高精度传感器 |
| 0.1 | 中 | 高 | MS901M (推荐) |
| 1.0 | 低 | 极高 | 噪声较大传感器 |

## 上位机调整方式

> **注意**: 卡尔曼滤波参数**只能在 Web Dashboard 的姿态页面中调整**，不推荐在下位机代码中硬编码修改。

### 姿态页面控制面板

| 功能 | 说明 |
|------|------|
| Q 值滑块 | 范围 0.0001 - 0.1，控制响应速度 |
| R 值滑块 | 范围 0.01 - 1.0，控制平滑度 |
| 通道选择 | 可分别调整 Roll/Pitch/Yaw/GyroX/GyroY/GyroZ |
| 实时预览 | 调整后立即查看滤波效果 |
| 复位按钮 | 恢复默认参数 (Q=0.001, R=0.1) |

### 调整流程

```
1. 连接设备，进入姿态页面
2. 选择需要调整的通道 (如 Roll)
3. 拖动 Q 值滑块，观察响应速度变化
4. 拖动 R 值滑块，观察平滑度变化
5. 找到最佳参数组合
6. 点击"应用参数"写入下位机
```

### 推荐参数预设

| 场景 | Q 值 | R 值 | 效果 |
|------|------|------|------|
| 水下稳定模式 | 0.001 | 0.1 | 平衡响应与平滑 |
| 静态测量模式 | 0.0001 | 0.5 | 极平滑，响应慢 |
| 动态跟踪模式 | 0.01 | 0.05 | 快速响应，略抖动 |

## 注意事项

1. **MS901M 已内置滤波** - 模块本身已进行卡尔曼滤波，本系统滤波为额外平滑
2. **参数需根据实际调整** - 不同应用场景可能需要不同的 Q/R 值
3. **浮点运算** - 依赖 Cortex-M4 的 FPU 单元，确保编译器开启 FPU 支持
4. **初始化值** - 首次上电时滤波值从 0 开始收敛，约需 10-20 次迭代

## 故障排除

| 现象 | 可能原因 | 解决方案 |
|------|---------|---------|
| 滤波后数据滞后 | Q 值过小 | 增大 Q 值 |
| 滤波效果不明显 | R 值过大 | 减小 R 值 |
| 数据发散 | 参数不合理 | 检查 Q/R 比值 |
