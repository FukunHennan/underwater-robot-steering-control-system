# 卡尔曼滤波器

## 概述

系统使用**六通道线性卡尔曼滤波器**处理姿态与陀螺仪数据：

| 通道 | 输入 | Q 默认值 | R 默认值 |
|------|------|---------|---------|
| Roll | 姿态角 + GyroX预测 | 0.001 | 0.1 |
| Pitch | 姿态角 + GyroY预测 | 0.001 | 0.1 |
| Yaw | 姿态角 + GyroZ预测 | 0.001 | 0.1 |
| GyroX | 陀螺仪X轴 | 0.01 | 0.05 |
| GyroY | 陀螺仪Y轴 | 0.01 | 0.05 |
| GyroZ | 陀螺仪Z轴 | 0.01 | 0.05 |

---

## 算法流程

```
预测阶段:
  x_pred = x + gyro × dt       (用陀螺仪积分预测姿态)
  P_pred = P + Q

更新阶段:
  K = P_pred / (P_pred + R)    (卡尔曼增益)
  x = x_pred + K × (z - x_pred)  (z 为传感器测量值)
  P = (1 - K) × P_pred
```

**Yaw 特殊处理**: Yaw 在 ±180° 处存在跳变，滤波前对测量值进行解绕：
```c
while (yaw_meas - kf_yaw.x >  180.0f) yaw_meas -= 360.0f;
while (yaw_meas - kf_yaw.x < -180.0f) yaw_meas += 360.0f;
// 滤波后再归一化回 [-180, 180]
```

---

## 参数调节指南

| 参数 | 增大效果 | 减小效果 |
|------|---------|---------|
| **Q (过程噪声)** | 响应更快，更跟随陀螺仪 | 响应更慢，更平滑 |
| **R (测量噪声)** | 更信任预测（陀螺仪），平滑但漂移 | 更信任测量（姿态角），跟随但噪声大 |

**推荐范围**:
- Q: 0.0001 ~ 0.1
- R: 0.01 ~ 1.0

---

## Modbus 实时调节

参数通过上位机**高级页面**实时下发，地址 0x0086–0x009C（均为 float 类型）。

写入 `KALMAN_CMD = 0x5A5A` 可复位所有滤波器状态为 0。
