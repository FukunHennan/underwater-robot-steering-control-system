# STM32F407 引脚连接说明

本文档记录水下智能转向系统的硬件引脚分配和连接方式。

---

## 主控芯片

**型号**: STM32F407VET6
- 内核: ARM Cortex-M4
- 主频: 168 MHz
- Flash: 512 KB
- SRAM: 192 KB

---

## 红外遥控

| 功能 | 引脚 | 中断 | 说明 |
|------|------|------|------|
| IR_RX (DI2) | PE4 | EXTI4 | 红外接收 (NEC协议) |
| IR_TX (DI1) | PC5 | - | 红外发射 (NEC协议) |

**红外接收器连接**:
```
38kHz IR接收模块:
  VCC  → 3.3V 或 5V
  GND  → GND
  OUT  → PE4
```

**红外发射器连接**:
```
38kHz IR发射模块:
  VCC  → 3.3V 或 5V
  GND  → GND
  IN   → PC5
```

**NEC 协议说明**:
- 引导码: 9ms 高 + 4.5ms 低
- 地址: 8位 + 地址反码
- 命令: 8位 + 命令反码
- 数据"0": 560μs 高 + 560μs 低
- 数据"1": 560μs 高 + 1690μs 低
